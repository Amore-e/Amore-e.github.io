# 数组

数组是相同类型数据的有序集合。数组描述的是相同类型的若干数据，按照一定的先后顺序排列组合。其中每一个数据称为元素，每个元素都可以通过数组的索引或者下标来定位。

数组的特点

  • 数组本身是引用数据类型。而数组中的元素可以是任意类型。
  • 创建数组对象会在内存中开辟一块空间，他的空间是连续的。而数组名引用的是该连续空间的首地址。
  • 数组的长度，一旦确定就不能修改。
  • 数组中存储的数据类型必须是相同，不能出现混合类型。
  • 数组的长度属性： length 。
  • 数组的索引：从 0 开始连续计数，至 length-1。当数组的索引超过数组的长度，就会发生异常 ArrayIndexOfBoundsException 。

数组的分类

  • 维度：一维数组、二维数组、多维数组…
  • 数据类型：基本类型的数组、引用类型的数组

 

# 一维数组

数组的定义格式： 数据类型[] 变量名; / 数据类型 变量名[];

 

# 数组的创建和初始化

数组的初始化：java 中的数组必须进行进行初始化，才能使用。初始化是为数组中的元素分配内存空间，并为每个元素进行赋值。

数组的初始化方式：

  • 静态初始化
  • 动态初始化

动态初始化

动态初始化：数组元素的分配空间及赋值分开进行。
格式： 数据类型[] 变量名 = new 数据类型[数组长度]

静态初始化

静态初始化：在定义数组的同事就为数组元素分配空间并赋值。
格式： 数据类型[] 变量名 = new 数据类型[]{数据1，数据2...}

此时，在声明数组的时候不需要指定长度，系统会根据数据类型自动分配空间。
简化格式： 数据类型[] 变量名 = {数据1，数据2...}

数组的默认值：数组元素即使我们没有赋值，都存在一个默认值。
