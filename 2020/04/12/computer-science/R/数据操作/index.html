# 二、属性、类、模式、基本类型

attributes() 查看对象的属性。

class() 查看对象的类。

mode() 查看对象的存储类型。

storage.mode() 查询的是数据实际在内存中存储所用的方式。

typeof() 的输出结果和 storage.mode() 相同。

类型 class() MODE() TYPEOF() storage.mode()
逻辑型 logical logical logical logical
整数型 numeric numeric integer integer
实数型 numeric numeric double double
复数型 complex complex complex complex
字符型 character character character character
字节型 raw raw raw raw
列表 list list — —
数据框 data.frame list — —
矩阵 matrix — — —
数组 array — — —
因子 factor — — —
日期 Date — — —

  • 当 x 是单个值，或者向量的时候，

    返回的结果 class 和 mode 一致，如 numeric，character

  • 当 x 是其他情况（矩阵，数组，日期，因子），

    class 返回（matrix，array，Date，factor）

    mode 返回（x 中元素的类型 —— 在内存中的存储类型）

  • 当 x 是数据框的时候，

    class 返回 data.frame ，

    mode 返回 list。

  • 当 x 是列表的时候，

    class 和 mode 都返回 list。

  • 为何数据框和列表不再适用于第二条 mode 返回 x 中元素的类型了呢？

    • 因为数据框其实是列表的一种特殊情况。

    • list 和 data.frame 中的数据类型可以不一致，所以没法返回一个类型代表多种元素类型。

# 三、数据结构

  • # 向量

（1）向量是 R 语言中最基本的数据结构。

（2）使用赋值符号 “ <- ” 和组合函数 c() , 就能创建各种类型的向量。

（3）向量必须由类型相同的元素组成。

（4） length() 函数返回向量的长度。长度为 1 的向量为标量，是特殊的向量。

（5） attributes() 函数可以查看向量的属性，诸如类、名称、维度等信息。

（6） names() 函数用来给向量的元素命名。

  • # 矩阵

（1）矩阵，是用行和列对元素进行定位的二维数据结构。

（2）基本用法：

data : 可选的数据向量 (包括列表或表达式向量)。

nrow : 所要创建的矩阵的行数。

nlol : 所要创建的矩阵的列数。

byrow: 是否按照行来将数据填充进入矩阵。

dimnamens : 矩阵维度的名称。

（3）矩阵必须由类型相同的元素组成。

（4）元素的标记和访问：矩阵生成后， [行号，列号] 用来显示元素的位置。

  • 选取矩阵的某一个元素；

  • 选取矩阵某一行所有列的元素；

  • 选取矩阵多行多列的元素；eg: m[c(1,2),c(1,3)]

  • 剔除矩阵某行和某列，保留剩下的元素； m[-1,]、m[-1,-3]

（5）用向量生成矩阵：结合函数 cbind() 或 rbind() 。

  • cbind() : 将所需要的变量以列的方式组合。

  • rbind() : 将所需要的变量以行的方式组合。

（6）转置函数 t() :

（7）行和列的命名： rownames() 和 colnames()

  • # 数组

    （1）如果说矩阵是向量的推广，那么数组就是矩阵的推广。数组和矩阵 作为数据结构的主要特征就是具有维度。数组和矩阵就是添加了维 度属性的向量。

    （2）基本用法：

    data : 数组中的数据。

    dim : 所要创建的数组的维度。

    ​ a. 数组中用来确定元素位置下标的个数，即为维度。

    ​ b. dim() 函数返回各个维度的最大值。

    ​ c. 根据 “矩阵是赋予了维度属性的向量”，

    ​ 可应用于将向量转换成数组。

    1. 对 dim = length(data) 的理解:

    dimnames : 对数组的各个维度命名。

    （3）数组中的所有元素必须是同一类型的数据。
